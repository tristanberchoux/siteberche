New Feather Format for Data Frames

Roger Peng 2016/03/31

This past Tuesday, Hadley Wickham and Wes McKinney announced a new binary file format specifically for storing data frames. One thing that struck us was that, while R’s data frames and Python’s pandas data frames utilize different internal memory representations, the semantics of their user data types are mostly the same. In both R and pandas, data frames contain lists of named, equal-length columns, which can be numeric, boolean, and date-and-time, categorical (factors), or string.
